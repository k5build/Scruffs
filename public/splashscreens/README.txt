SPLASH SCREEN GENERATION
========================

Required splash screen dimensions (portrait only):

iPhone 14 Pro Max  -> 1290 x 2796 px -> iphone14-pro-max.png
iPhone 14 Pro      -> 1179 x 2556 px -> iphone14-pro.png
iPhone 14          -> 1170 x 2532 px -> iphone14.png
iPhone X/11/12/13  -> 1125 x 2436 px -> iphonex.png
iPhone XR/11       -> 828  x 1792 px -> iphonexr.png
iPhone 8 Plus      -> 1242 x 2208 px -> iphone8plus.png
iPhone 8           -> 750  x 1334 px -> iphone8.png

Design:
- Background: #F4F2EE (brand warm cream)
- Center logo: logo-icon-green.png at ~200px
- Brand name "SCRUFFS" below logo in #3A4F4A

Tools to generate:
- Use Figma (free) to design at each size and export as PNG
- OR use: https://progressier.com/pwa-screenshots-and-icons
- OR use: https://www.appicon.co for icons, Figma for splashscreens

After creating, place files in: public/splashscreens/
